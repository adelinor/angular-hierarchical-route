About

This is a sample application to illustrate the use of routing in the truly heroic AngularJS framework. To give this app a neat look, the great CSS Bootstrap is used.

This sample has three main routes:

  • /about: this page's content is static
  • /admin: this is a page to expand the list of predefined cities
  • /home: this allows a user to select a location*. The current weather then shows for the selected location. A user can from there navigate to the next 48 hours forecast.

    [*]: a location is selected by choosing first a country. This then brings the list of cities for that country in a second drop down.

This HTML layout is a modified version of the CSS Bootstrap example Stick footer with fixed navbar
